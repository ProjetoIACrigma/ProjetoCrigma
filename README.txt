Para que possamos verificar o funcionamento do programa, é necessário realizar o download dos arquivos presentes no repositório.

O funcionamento exige a instalação do programa python e algumas bibliotecas como piclke e pandas. 

Um servidor local deve ser executado para rodar o código na web, portanto, recomendamos que seja executado dentro do programa pycharm.

Após rodar o código e acessar o servidor web, basta digitar o(s) filme(s) para gerar recomendações em cima desses títulos.

O programa retornará uma lista de filmes que combinam com os títulos mencionados pelo usuário.
